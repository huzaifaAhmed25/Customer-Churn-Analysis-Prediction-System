# Customer-Churn-Analysis-Prediction-System
📡 Churn Intelligence Dashboard is a Streamlit-based machine learning project that analyzes telecom customer data and predicts churn risk. It uses Logistic Regression and Random Forest models, with interactive visualizations, EDA, and real-time prediction to help businesses understand customer behavior and improve retention strategies.
✨ Overview

Customer churn is one of the biggest challenges in subscription-based businesses.
This project helps companies understand why customers leave and who is likely to leave next using data analysis and machine learning.

The system combines data visualization + machine learning + interactive dashboard to deliver actionable insights.

🎯 Key Highlights
✔ Understand customer behavior patterns
✔ Identify factors causing churn
✔ Predict churn probability in real-time
✔ Compare ML models performance
✔ Interactive dashboard for easy exploration

📊 Features
📈 Beautiful interactive data visualizations (Plotly)
🔍 Exploratory Data Analysis (EDA)
🧠 Machine Learning prediction system
⚖️ Model comparison (Logistic Regression vs Random Forest)

🤖 Machine Learning Models
Logistic Regression (Baseline model)
Random Forest Classifier (Advanced ensemble model)
Both models are trained on cleaned and preprocessed telecom data to ensure accurate predictions.

🛠️ Tech Stack

💡 How It Works
Upload telecom dataset
Data is cleaned & processed
Models are trained automatically
Dashboard shows insights & patterns

System predicts churn risk instantly
